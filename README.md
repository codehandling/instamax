Benefits
========

4.0 features
- Displays all photos of the User
- Grey and White Skins available
- Works on any device size (responsive + fluid)
- Improved lightbox popup for images (responsive + fluid)
- Optimized code overload (minified JS+CSS ~ 50KB)
